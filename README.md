# books-site

# Initialization of the project
  There are two independent projects in this repo
  Front was developed with node 14.17.0, yarn 1.22.17
  You must initialize them separately.
  Clone and run on each:
  - `yarn`  /  `npm install`

## For development

  - `yarn start`  /  `npm start`

## To build production version

  - `yarn build`  /  `npm run build`
